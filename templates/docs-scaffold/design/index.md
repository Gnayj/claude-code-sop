# Design docs index

Module-level feature designs and architecture live here, one file per design:
`docs/design/<module>/<design-id>-design.md` (start from `docs/plans/_template-design.txt`).

A design doc is the round contract for single-round (N=1) implements (see the SOP / collaboration
protocol). Record its review chain at the top and the implementation record + `code-home:` at the end.
