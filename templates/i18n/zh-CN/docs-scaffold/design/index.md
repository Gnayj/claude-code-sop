# 设计文档索引

`docs/design/<module>/<design-id>-design.md`（从 `docs/plans/_template-design.txt` 起手）。

对单轮（N=1）implement，design 文档即该轮契约（见 SOP / 协作协议）。在顶部记录其 review chain，
在末尾记录实施记录 + `code-home:`。
